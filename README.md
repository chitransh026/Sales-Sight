## 🎯 Overview

Sales-Sight is a modern sales analytics dashboard that provides real-time insights into sales metrics, customer data, and performance trends. It offers an intuitive interface for sales teams and managers to monitor key performance indicators (KPIs) and make data-driven decisions.

## ✨ Features

- **Sales Dashboard** - Real-time overview of sales metrics and KPIs
- **Performance Analytics** - Track sales trends and patterns
- **Customer Management** - View and manage customer information
- **Sales Reports** - Generate comprehensive sales reports
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **User-Friendly Interface** - Intuitive navigation and controls

## 🛠️ Technologies Used

- **HTML5** - Structure and markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and dynamic functionality
- **Flask**- A lightweight web development pacakage
- **Numpy**- A Python library to calculate Numerical methods
- **Pandas**- Used for Data visualisation in Tables(dataframe)
- **Plotly**- For Plots 

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/chitransh026/Sales-Sight.git
   cd Sales-Sight
   ```

2. **Open in your browser:**
   - Simply open the main HTML file in your web browser
   - Or use a local development server for better experience:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (if you have http-server installed)
     http-server
     ```

3. **Access the application:**
   - Navigate to `http://localhost:8000` (or the port shown in your terminal)

## 🚀 Usage

1. Open the application in your web browser
2. Navigate through the dashboard using the main menu
3. View sales metrics and analytics on the dashboard
4. Use filters and date ranges to customize your view
5. Generate and export reports as needed
